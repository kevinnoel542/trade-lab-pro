                        💡 USEFUL COMMANDS

Check if services are running:
  curl http://localhost:3001/api/health
  curl http://localhost:8081

View logs:
  tail -f /tmp/tradelab-backend.log
  tail -f /tmp/tradelab-frontend.log

Check database:
  mysql -u tradevault -p tradevault
  SELECT COUNT(*) FROM trades;

Kill running processes:
  pkill -f "node server"    (kill backend)
  pkill -f "vite"           (kill frontend)

Backup database:
  mysqldump -u tradevault -p tradevault > backup.sql

Restore database:
  mysql -u tradevault -p tradevault < backup.sql
