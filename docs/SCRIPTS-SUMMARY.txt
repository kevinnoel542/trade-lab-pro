================================================================================
                         TROUBLESHOOTING QUICK FIX
================================================================================

MySQL not running?
   Mac:   brew services start mysql
   Linux: sudo systemctl start mysql
   Win:   Search "Services", start MySQL80

Port in use?
   Change in configure-env.sh/bat option 1

npm install errors?
   rm -rf node_modules && npm install

Permission denied on scripts?
   chmod +x setup-offline.sh start-servers.sh configure-env.sh
